🎯 接下来的 3 个步骤：

第 1 步：推送到 GitHub
─────────────────────────────────────────────────────────────
如果还没有 GitHub 仓库：
1. 访问 https://github.com/new 创建新仓库
2. 执行命令：
   git remote add origin https://github.com/你的用户名/你的仓库名.git
   git push -u origin main

如果已有仓库：
   git push origin main

═══════════════════════════════════════════════════════════════

第 2 步：部署后端到 Railway（5分钟）
─────────────────────────────────────────────────────────────
1. 访问 https://railway.app/
2. 用 GitHub 登录
3. New Project → Deploy from GitHub repo → 选择你的仓库
4. 等待自动构建完成
5. 点击 "+ New" → Database → Add MySQL
6. 点击后端服务 → Variables → Raw Editor，粘贴：

SPRING_DATASOURCE_URL=jdbc:mysql://${{MySQL.MYSQL_HOST}}:${{MySQL.MYSQL_PORT}}/${{MySQL.MYSQL_DATABASE}}?useSSL=false&serverTimezone=Asia/Shanghai&characterEncoding=utf8&allowPublicKeyRetrieval=true
SPRING_DATASOURCE_USERNAME=${{MySQL.MYSQL_USER}}
SPRING_DATASOURCE_PASSWORD=${{MySQL.MYSQL_PASSWORD}}
SERVER_PORT=8080
COMPILER_GCC_PATH=gcc

7. Settings → Domains → Generate Domain
8. 复制后端 URL（例如：your-app.up.railway.app）
9. 点击 MySQL → Data → Query，导入 springboot/src/main/resources/sql/init.sql

═══════════════════════════════════════════════════════════════

第 3 步：部署前端到 Vercel（3分钟）
─────────────────────────────────────────────────────────────
1. 访问 https://vercel.com/
2. 用 GitHub 登录
3. Add New → Project → Import 你的仓库
4. 配置：
   - Framework Preset: Vite
   - Root Directory: vue
   - Build Command: npm run build
   - Output Directory: dist
5. Environment Variables：
   - Name: VITE_API_BASE_URL
   - Value: https://your-app.up.railway.app（你的 Railway 后端地址）
6. 点击 Deploy
7. 等待完成，复制前端 URL
