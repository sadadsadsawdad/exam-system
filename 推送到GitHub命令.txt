# 推送代码到 GitHub 的命令

# 方式一：如果是新仓库（推荐）
# 复制 GitHub 显示的命令，类似：

git remote add origin https://github.com/你的用户名/exam-system.git
git branch -M main
git push -u origin main

# 或者使用 SSH（如果配置了 SSH key）：
git remote add origin git@github.com:你的用户名/exam-system.git
git branch -M main
git push -u origin main

---

# 方式二：如果已经有 remote，需要先删除
git remote remove origin
git remote add origin https://github.com/你的用户名/exam-system.git
git branch -M main
git push -u origin main

---

# 检查当前分支
git branch

# 如果当前分支是 master，需要重命名为 main
git branch -M main

# 查看远程仓库
git remote -v

# 推送代码
git push -u origin main

---

# 如果推送时需要登录
# 会弹出 GitHub 登录窗口，使用你的 GitHub 账号登录即可

---

# 推送成功后，访问你的 GitHub 仓库
# https://github.com/你的用户名/exam-system

# 确认所有文件都已上传
