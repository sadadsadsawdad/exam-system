# 在线考试系统部署文档

## 项目概述

这是一个基于 Spring Boot + Vue 3 的前后端分离在线考试系统。

- 后端：Spring Boot 3.4.12 + MyBatis + MySQL
- 前端：Vue 3 + Element Plus + Vite
- 数据库：MySQL 8.0+
- Java 版本：JDK 17

---

## 一、服务器环境准备

### 1.1 基础环境要求

- 操作系统：Linux (推荐 Ubuntu 20.04+ 或 CentOS 7+)
- 内存：至少 2GB RAM
- 磁盘：至少 10GB 可用空间

### 1.2 安装必要软件

#### 安装 JDK 17

```bash
# Ubuntu/Debian
sudo apt update
sudo apt install openjdk-17-jdk -y

# CentOS/RHEL
sudo yum install java-17-openjdk-devel -y

# 验证安装
java -version
```

#### 安装 MySQL 8.0

```bash
# Ubuntu/Debian
sudo apt install mysql-server -y

# CentOS/RHEL
sudo yum install mysql-server -y

# 启动 MySQL
sudo systemctl start mysql
sudo systemctl enable mysql

# 安全配置
sudo mysql_secure_installation
```

#### 安装 Nginx

```bash
# Ubuntu/Debian
sudo apt install nginx -y

# CentOS/RHEL
sudo yum install nginx -y

# 启动 Nginx
sudo systemctl start nginx
sudo systemctl enable nginx
```

#### 安装 Maven (用于构建后端)

```bash
# Ubuntu/Debian
sudo apt install maven -y

# CentOS/RHEL
sudo yum install maven -y

# 验证安装
mvn -version
```

#### 安装 Node.js (用于构建前端)

```bash
# 使用 NodeSource 安装 Node.js 20.x
curl -fsSL https://deb.nodesource.com/setup_20.x | sudo -E bash -

# 或使用 nvm 安装

```

#### 安装 GCC (用于 C 代码编译功能)

```bash

sudo apt install build-essential -y

# CentOS/RHEL
sudo yum groupinstall "Development Tools" -y

# 验证安装
gcc --version
```

## 二、数据库配置

### 2.1 创建数据库和用户

```bash
# 登录 MySQL
sudo mysql -u root -p

# 在 MySQL 中执行以下命令
```

```sql
-- 创建数据库
CREATE DATABASE exam CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;

-- 创建用户（请修改密码）
CREATE USER 'examuser'@'localhost' IDENTIFIED BY '你的强密码';
```

```bash
# 导入数据库结构和初始数据
```

## 三、后端部署

### 3.1 修改配置文件

编辑 `springboot/src/main/resources/application.properties`：

```properties
# 数据库配置（修改为实际配置）
spring.datasource.url=jdbc:mysql://localhost:3306/exam?useSSL=false&serverTimezone=Asia/Shanghai&characterEncoding=utf8&allowPublicKeyRetrieval=true
spring.datasource.username=examuser
spring.datasource.password=你的数据库密码
spring.datasource.driver-class-name=com.mysql.cj.jdbc.Driver

# MyBatis 配置
mybatis.mapper-locations=classpath:mappers/*.xml
mybatis.configuration.map-underscore-to-camel-case=true

# 服务端口
server.port=8081

# GCC 编译器路径（Linux 默认）
compiler.gcc.path=gcc
```

### 3.2 构建后端项目

```bash
# 进入后端目录
cd springboot

# 清理并打包
mvn clean package -DskipTests

# 打包完成后，jar 文件位于 target/springboot-0.0.1-SNAPSHOT.jar
```

### 3.3 部署后端服务

#### 方式一：使用 systemd 服务（推荐）

创建服务文件：

```bash
sudo nano /etc/systemd/system/exam-backend.service
```

添加以下内容：

```ini
[Unit]
Description=Exam System Backend Service
After=mysql.service

[Service]
Type=simple
User=www-data
WorkingDirectory=/opt/exam-backend
ExecStart=/usr/bin/java -jar /opt/exam-backend/springboot-0.0.1-SNAPSHOT.jar
Restart=on-failure
RestartSec=10
StandardOutput=journal
StandardError=journal

[Install]
WantedBy=multi-user.target
```

部署步骤：

```bash
# 创建部署目录
sudo mkdir -p /opt/exam-backend

# 复制 jar 文件
sudo cp target/springboot-0.0.1-SNAPSHOT.jar /opt/exam-backend/

# 设置权限
sudo chown -R www-data:www-data /opt/exam-backend

# 重载 systemd
sudo systemctl daemon-reload

# 启动服务
sudo systemctl start exam-backend

# 设置开机自启
sudo systemctl enable exam-backend

# 查看服务状态
sudo systemctl status exam-backend

# 查看日志
sudo journalctl -u exam-backend -f
```

#### 方式二：使用 nohup 后台运行

```bash
# 创建部署目录
mkdir -p /opt/exam-backend
cp target/springboot-0.0.1-SNAPSHOT.jar /opt/exam-backend/

# 后台运行
cd /opt/exam-backend
nohup java -jar springboot-0.0.1-SNAPSHOT.jar > app.log 2>&1 &

# 查看日志
tail -f app.log
```

## 四、前端部署

### 4.1 修改 API 地址

编辑 `vue/vite.config.js`，添加生产环境 API 配置：

```javascript
import { fileURLToPath, URL } from 'node:url'
import { defineConfig } from 'vite'
import vue from '@vitejs/plugin-vue'

export default defineConfig({
  plugins: [vue()],
  resolve: {
    alias: {
      '@': fileURLToPath(new URL('./src', import.meta.url))
    },
  },
  server: {
    proxy: {
      '/api': {
        target: 'http://localhost:8081',
        changeOrigin: true
      }
    }
  },
  build: {
    chunkSizeWarningLimit: 1500,
    rollupOptions: {
      output: {
        manualChunks: {
          'element-plus': ['element-plus'],
          'echarts': ['echarts'],
          'vue-vendor': ['vue', 'vue-router']
        }
      }
    }
  }
})
```

### 4.2 构建前端项目

```bash
# 进入前端目录
cd vue

# 安装依赖
npm install

# 构建生产版本
npm run build

# 构建完成后，静态文件位于 dist 目录
```

### 4.3 配置 Nginx

创建 Nginx 配置文件：

```bash
sudo nano /etc/nginx/sites-available/exam-system
```

添加以下内容：

```nginx
server {
    listen 80;
    server_name your-domain.com;  # 修改为你的域名或服务器 IP

    # 前端静态文件
    root /var/www/exam-frontend;
    index index.html;

    # 前端路由支持
    location / {
        try_files $uri $uri/ /index.html;
    }

    # 后端 API 代理
    location /api/ {
        proxy_pass http://localhost:8081/;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
        proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
        proxy_set_header X-Forwarded-Proto $scheme;
        
        # 超时设置
        proxy_connect_timeout 60s;
        proxy_send_timeout 60s;
        proxy_read_timeout 60s;
    }

    # 静态资源缓存
    location ~* \.(js|css|png|jpg|jpeg|gif|ico|svg|woff|woff2|ttf|eot)$ {
        expires 1y;
        add_header Cache-Control "public, immutable";
    }

    # Gzip 压缩
    gzip on;
    gzip_vary on;
    gzip_min_length 1024;
    gzip_types text/plain text/css text/xml text/javascript application/x-javascript application/xml+rss application/javascript application/json;
}
```

部署前端文件：

```bash
# 创建前端部署目录
sudo mkdir -p /var/www/exam-frontend

# 复制构建文件
sudo cp -r vue/dist/* /var/www/exam-frontend/

# 设置权限
sudo chown -R www-data:www-data /var/www/exam-frontend

# 启用站点配置
sudo ln -s /etc/nginx/sites-available/exam-system /etc/nginx/sites-enabled/

# 测试 Nginx 配置
sudo nginx -t

# 重启 Nginx
sudo systemctl restart nginx
```

### 8.3 更新部署

#### 更新后端

```bash
# 构建新版本
cd springboot
mvn clean package -DskipTests

# 停止服务
sudo systemctl stop exam-backend

# 备份旧版本
sudo cp /opt/exam-backend/springboot-0.0.1-SNAPSHOT.jar /opt/exam-backend/springboot-0.0.1-SNAPSHOT.jar.bak

# 复制新版本
sudo cp target/springboot-0.0.1-SNAPSHOT.jar /opt/exam-backend/

# 启动服务
sudo systemctl start exam-backend
```

#### 更新前端

```bash
# 构建新版本
cd vue
npm run build

# 备份旧版本
sudo cp -r /var/www/exam-frontend /var/www/exam-frontend.bak

# 复制新版本
sudo rm -rf /var/www/exam-frontend/*
sudo cp -r dist/* /var/www/exam-frontend/

# 重启 Nginx
sudo systemctl restart nginx
```

### 8.4 数据库备份

```bash
# 创建备份脚本
sudo nano /opt/backup-db.sh
```

添加以下内容：

```bash
#!/bin/bash
BACKUP_DIR="/opt/backups"
DATE=$(date +%Y%m%d_%H%M%S)
mkdir -p $BACKUP_DIR

mysqldump -u examuser -p'你的密码' exam > $BACKUP_DIR/exam_$DATE.sql

# 保留最近 7 天的备份
find $BACKUP_DIR -name "exam_*.sql" -mtime +7 -delete

echo "Backup completed: exam_$DATE.sql"
```

设置定时备份：

```bash
# 添加执行权限
sudo chmod +x /opt/backup-db.sh

# 添加到 crontab（每天凌晨 2 点备份）
sudo crontab -e

# 添加以下行
0 2 * * * /opt/backup-db.sh >> /var/log/db-backup.log 2>&1
```

## 九、性能优化建议

### 9.1 JVM 参数优化

修改 systemd 服务文件中的 ExecStart：

```ini
ExecStart=/usr/bin/java -Xms512m -Xmx1024m -XX:+UseG1GC -jar /opt/exam-backend/springboot-0.0.1-SNAPSHOT.jar
```

### 9.2 MySQL 优化

编辑 `/etc/mysql/mysql.conf.d/mysqld.cnf`：

```ini
[mysqld]
max_connections = 200
innodb_buffer_pool_size = 512M
innodb_log_file_size = 128M
query_cache_size = 0
query_cache_type = 0
```

### 9.3 Nginx 优化

在 nginx.conf 中添加：

```nginx
worker_processes auto;
worker_connections 1024;
keepalive_timeout 65;
client_max_body_size 20M;
```

## 十、故障排查

### 常见问题

1. **后端无法启动**
   - 检查 Java 版本：`java -version`
   - 检查端口占用：`sudo netstat -tlnp | grep 8081`
   - 查看日志：`sudo journalctl -u exam-backend -n 100`

2. **数据库连接失败**
   - 检查 MySQL 状态：`sudo systemctl status mysql`
   - 验证数据库连接：`mysql -u examuser -p`
   - 检查防火墙规则

3. **前端页面无法访问**
   - 检查 Nginx 状态：`sudo systemctl status nginx`
   - 测试配置：`sudo nginx -t`
   - 查看错误日志：`sudo tail -f /var/log/nginx/error.log`

4. **API 请求失败**
   - 检查后端服务是否运行
   - 验证 Nginx 代理配置
   - 检查浏览器控制台网络请求
